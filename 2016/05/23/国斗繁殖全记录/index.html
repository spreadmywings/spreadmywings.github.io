国斗繁殖全记录

发表于 2016-05-23   |   分类于 乐活   |     |   阅读次数

开篇

小时候会养些小动物什么的，尤其对这种小鱼情有独钟。长大了才知道这鱼叫中国斗鱼，我们那地方捞到的一般都是普通叉尾的品种，简称普叉。斑斓的色彩优雅的姿态，这对小朋友来说是多大的诱惑力，如今难得能再捞上几条全部赠予友人，又适逢该鱼种交配时节，于是一同拾起儿时养小鱼的乐趣，在朋友精心筹备之下第一次见证了国斗繁殖的全过程。

第1天

国斗的繁殖过程的确很特别，也很有趣。雄鱼发情之后会在水面吐出一大堆泡泡（5月20日）。
此时只有同样是发情并且“身怀六甲”的雌鱼，并且身形大小跟雄鱼相匹配才能进入雄鱼视野，否则管你有没意思，公的母的照样赶跑，要是地方小没地躲那就悲剧了，送去的9条国斗，光打架受伤最后染上水霉病挂掉的就没了5条。

第2天

当抢情投意合的两方合抱过之后（这过程朋友没记录到，可能都躲到泡泡底私混去了），雌鱼产卵，雄鱼则将卵衔着放入泡泡中，于是就看到下图中有些略带黄色，明显更加晶莹剔透的泡泡（5月21日早上）。
根据网上说法，卵会在一到两天内孵化。5月21日中午观察到水面的泡泡没那么厚了，傍晚时候气泡继续消散，但并没有发现鱼苗。可能是鱼苗太小了，肉眼太难察觉，还以为出了什么乱子，继续观察。

第3天

第3天早上（5月22日），水面上能看到细微的小黑点在晃动，仔细观察终于发现小鱼苗的身影，小得像个芝麻一样还真难发现。看你们能否看得出来？来张红色区域放大图，这样看起来就明显多了。小鱼苗刚出世，泡泡上还有卵黄，小生命还得依靠它们生存好几天。这时候雄鱼就尽显奶爸本色了，除了警惕地四处巡游保护幼苗安全还得时刻把游离泡泡的小鱼苗用口衔回泡泡附近。
直到中午时分，小鱼苗的身体已经黑了很多，已经很明显能看到它们的存在，就像个小蚊虫一样，并且也活跃了很多。
傍晚再去观察，小鱼苗的身形已有明显增大，看着好大一群粘在泡泡边上。放大了再看。半透明的身子，身上黑色的线条显得格外醒目，这是骨头和内脏么？

结语

用朋友的话来说，这次经历是相当的过瘾！小时候养鱼还没看到过国斗生小鱼的，还是第一次感受到全过程。来张绝世奶爸出场的温馨场面。现在问题来了，上百近两百条的鱼苗，怎么照顾怎么处置都是压力呀。今天朋友说准备买个收纳箱当鱼缸照顾这班小生命了…
